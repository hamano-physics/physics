無限遠で運動エネルギーが 0 以上なら、物体は地球から飛び去れます。出発点と無限遠で 力学的エネルギーが保存されます（位置エネルギーの基準は無限遠）。
½mv₀² − GMm/R = ½mv²(∞) + 0
STEP 2　最小の速さ v₀
無限遠での速さがちょうど 0 になるときが最小。
½mv₀² − GMm/R = 0　→　v₀² = 2GM/R
STEP 3　g で表す
GM = gR² を代入すると、
v₂ = √(2gR) = √(2×9.8×6.4×10⁶) ≒ 11.2 km/s
第2宇宙速度 v₂＝√(2gR) は約 11.2 km/s。第1宇宙速度の √2 倍です。これより遅いと いつか地球に戻り、これ以上なら無限遠まで飛び去ります。
